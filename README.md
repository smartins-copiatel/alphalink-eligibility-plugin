# Alphalink Eligibility Plugin

Plugin WordPress d'éligibilité Fibre/ADSL avec API Alphalink

## Version
3.2.0 - Stable (8 mars 2026)

## Installation
1. Téléchargez les fichiers
2. Uploadez dans `/wp-content/plugins/alphalink-eligibility/`
3. Activez le plugin dans WordPress
4. Ajoutez le shortcode `[alphalink_eligibility]` dans une page

## Fichiers
- `alphalink-eligibility.php` - Plugin principal
- `script.js` - JavaScript
- `style.css` - Styles CSS

## Configuration
Allez dans Alphalink Cache → Configuration API pour configurer vos identifiants Alphalink.

## Fonctionnalités
✅ Recherche d'adresse intelligente
✅ Détection automatique du numéro
✅ Carte interactive Leaflet
✅ Infos réseau (PM, NRA, distance)
✅ Affichage des offres par fournisseur
✅ Responsive mobile
