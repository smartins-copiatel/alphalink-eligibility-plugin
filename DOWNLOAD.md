## Version Stable - 8 Mars 2026

Les fichiers sont trop volumineux pour être hébergés directement sur GitHub.

### Fichiers à Télécharger

Tous les fichiers sont disponibles dans la conversation Limova.

**Copier-coller depuis Limova :**

1. **alphalink-eligibility.php** (280 lignes)
2. **script.js** (650 lignes) 
3. **style.css** (550 lignes)

### Installation

```bash
# Structure des dossiers
wp-content/plugins/
├── alphalink-eligibility/
│   ├── alphalink-eligibility.php
│   ├── script.js
│   └── style.css
```

### Activation
1. Uploadez les fichiers dans le dossier ci-dessus
2. Allez dans Extensions → Extensions installées
3. Activez "Alphalink Eligibility Form"
4. Ajoutez `[alphalink_eligibility]` dans une page
